# 🌱 GreenRouteAI  
**An Adaptive Multi-Objective Green Route Recommendation System**

## 📌 Overview  
GreenRouteAI is an AI-based routing framework designed for sustainable and health-aware urban mobility. Unlike traditional navigation systems that optimize only time or distance, this system integrates air quality (AQI), road networks, and vehicle emission models to recommend routes that minimize pollution exposure, emissions, and travel time.

---

## 🚀 Features  
- Multi-objective route optimization (time, emissions, exposure)  
- Real-time AQI-based pollution-aware routing  
- Vehicle-specific emission estimation using machine learning  
- Graph-based road network analysis  
- AQI heatmaps and route comparison visualization  
- Promotes eco-friendly and health-conscious travel  

---

## 🧠 System Architecture  
Pollution-Mapped Road Network → Distance & Time Calculation → Exposure & Emission Modeling → Cost Function → Routing Algorithm → Route Output + Visualization  

---

## 🛠️ Technologies Used  
- Python  
- Machine Learning (Scikit-learn / TensorFlow)  
- OpenStreetMap (OSM)  
- GeoPandas, NetworkX  
- AQI Data APIs  
- Data Visualization Tools  

---

## 📊 Key Results  
- Achieved **6.58% reduction in pollution exposure** compared to fastest routes  
- Balanced travel efficiency with environmental impact  
- Improved decision-making using route visualization
